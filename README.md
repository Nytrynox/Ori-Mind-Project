# Ori Mind Project

## Overview
Ori Mind is a cognitive enhancement tool designed to map and visualize complex thought processes. It helps users organize ideas, brainstorm effectively, and structure large projects using a dynamic, mind-mapping interface.

## Features
-   **Dynamic Mapping**: Create and link nodes to represent connected thoughts.
-   **Visual Organization**: Color-coded branches and intuitive layout tools.
-   **Export Options**: Save maps as images or structured data files.
-   **Real-time Collaboration**: (Planned) Multiple users editing the same map.

## Technology Stack
-   **Frontend**: JavaScript, HTML Canvas.
-   **Libraries**: D3.js for data visualization.

## Usage Flow
1.  **Start**: Create a central topic node.
2.  **Branch**: Add sub-topics and connect them to the center.
3.  **Detail**: Attach notes or files to specific nodes.
4.  **Visualize**: Zoom and pan to explore the full idea map.

## Quick Start
```bash
# Clone the repository
git clone https://github.com/Nytrynox/Ori-Mind-Project.git

# Launch
open index.html
```
